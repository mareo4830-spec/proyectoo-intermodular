¿En qué consiste el proyecto?

El proyecto tiene como objetivo implementar un Sistema de Control de Acceso Digital en una oficina, empleando una Raspberry Pi 4 como unidad central de gestión. Este sistema permitirá administrar de manera segura el ingreso y salida de personal mediante tarjetas RFID o códigos digitales.

Además, el sistema estará conectado a la red de la empresa y contará con un panel de administración accesible vía web. Desde ese panel será posible:

  • Registrar y eliminar usuarios autorizados.
  • Consultar el historial de accesos.
  • Configurar diferentes niveles de permisos.

Objetivos principales

  • Aumentar la seguridad en la oficina controlando quién accede a las instalaciones.
  • Utilizar software libre y hardware de bajo costo para facilitar la replicación del sistema.
  • Generar un sistema escalable, adaptable a diferentes entornos empresariales.
  • Desarrollar documentación técnica y manuales de uso para garantizar el mantenimiento a largo plazo.

Metodología

La metodología se basa en fases sucesivas:

  1. Análisis: Identificación de requisitos y definición del hardware/software necesario.
  2. Diseño: Elaboración de diagramas de red y arquitectura del sistema.
  3. Implementación: Configuración de Raspberry Pi, instalación de paquetes y desarrollo del software.
  4. Pruebas: Validación del sistema en escenarios reales y ajuste de parámetros.
  5. Documentación y entrega: Elaboración del informe final y presentación del proyecto.
